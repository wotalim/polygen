# Polygen
Inspired from [OBTGEN](https://github.com/fikriomar16/obtgen)


## Installation
```sh
 $ git clone https://github.com/wotalim/polygen.git
 $ cd polygen/
 $ cp -r polybar/ polygen ~/.local/bin/
```

## Usage
```sh
 $ polygen --wb
 
 or
 
 $ cd ~/.local/bin/
 $ ./polygen --wb
```
## Help
```sh
--wb      generate polybar color from pywal cache
--help    show help
```
